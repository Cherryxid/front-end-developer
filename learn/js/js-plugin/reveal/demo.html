自定义某张幻灯的样式设置

在section上增加data-state="自定义类名"，该页面的html元素的类名会加上"data-state"的值

支持页面出现时自定义行为。

在section添加data-state="自定义事件名"，在脚本中绑定事件处理函数。例如 


Reveal.addEventListener( 'customevent', function() {
	//自定义行为
} );

支持背景色和背景图片的自定义
