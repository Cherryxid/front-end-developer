# HTML 字符实体
在 HTML 中，某些字符是预留的。

在 HTML 中不能使用小于号（<）和大于号（>），这是因为浏览器会误认为它们是标签。

如果希望正确地显示预留字符，我们必须在 HTML 源代码中使用字符实体（character entities）。

显示字符实体可用实体名称或实体编号。

## 常用的字符实体及名称
* `空格`  `&nbsp;
* `<` `&lt;`
* `>` `&gt;`
* `&` `&amp;`
* `"` `&quot;`
* `'` `&apos;` (IE不支持)
* `©` `&copy;` 版权
* `®` `&reg;` 注册商标
* `×` `&times;`
